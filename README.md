## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project Initialization
- Node.js version: 20.14.0
- NestJS CLI version: 10.4.9

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
